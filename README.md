# Clankier Chainsaw
A fast-paced 2D survival game where you repel waves of incoming enemies by swinging your weapon. Visit the Shop between waves to spend your collected coins, grab random power-ups, and upgrade your robot.

## Features
Slash through enemies using keys 1–3 to perform different weapon maneuvers (sweep / flip).
Dozens of random upgrades across 3 distinct tiers—every run offers a unique build.
Scaling difficulty with increasingly tougher waves and diverse enemy types.
Snappy roguelike flow designed for that "just one more round" feeling.

Developers: Aziz Önder, Aleksey Khlopkhov
Artists: Efe Burak Vuran,  Merve, Elif Mina
